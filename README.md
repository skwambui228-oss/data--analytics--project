# 🛒 Customer Orders Analysis

## Project Overview
An end-to-end SQL data analytics project analyzing 
customer orders for a mock e-commerce store based in Kenya.

## Tools Used
- SQL (Supabase/PostgreSQL)
- Microsoft Excel

## Database Schema
- customers — customer details
- products — product catalog
- orders — order records
- order_items — individual order line items

## Key Analysis Questions
1. Who are the top 10 customers by total spend?
2. Which products sell most in Q4?
3. What is the monthly revenue trend?
4. Which product category generates most revenue?
5. Which cities generate the most orders?

## Key Findings
- Alice Mwangi is the top customer with KSh 141,000 in spend
- Electronics drives 90.5% of total revenue
- October is the strongest month with KSh 179,500 in revenue
- Laptops are the best performing product in Q4
- Nairobi leads all cities with KSh 154,250 in revenue
